Frequently Asked Questions

Bagaimana cara upgrade ke paket Premium? +
Anda dapat upgrade ke paket Premium kapan saja melalui dashboard akun Anda. Klik tombol "Upgrade" dan pilih paket yang diinginkan. Pembayaran dapat dilakukan melalui berbagai metode pembayaran yang tersedia.
Apakah ada kontrak jangka panjang? +
Tidak ada kontrak jangka panjang. Anda dapat membatalkan berlangganan kapan saja tanpa penalti. Pembayaran dilakukan bulanan dan dapat dihentikan setiap saat.
Bagaimana jika saya ingin downgrade? +
Anda dapat downgrade ke paket yang lebih rendah kapan saja. Perubahan akan berlaku pada periode billing berikutnya. Data yang sudah tersimpan akan tetap aman sesuai dengan kebijakan penyimpanan paket baru.
Apakah data saya aman? +
Ya, keamanan data adalah prioritas utama kami. Semua data dienkripsi dan disimpan dengan standar keamanan tinggi. Kami mematuhi regulasi privasi data internasional dan tidak akan membagikan data Anda kepada pihak ketiga.
Apakah tersedia trial untuk paket Premium? +
Ya, kami menyediakan trial gratis 14 hari untuk paket Premium. Anda dapat mencoba semua fitur premium tanpa perlu memasukkan informasi kartu kredit. Setelah trial berakhir, Anda dapat memilih untuk melanjutkan berlangganan atau kembali ke paket Basic.
